## What it does

Applies a system prompt prefix that instructs the AI to respond entirely in pirate dialect ‚ÄĒ "arr"s, nautical metaphors, and all. Factual accuracy is preserved; only the delivery changes.

## Usage

1. Install the plugin from the Marketplace.
2. In the **Installed** tab, click **Set active** next to Pirate Mode.
3. All subsequent responses will be delivered in pirate speak.
4. Click **Active** again to deactivate.

> **User:** What is machine learning?

## Notes

- Only one style plugin can be active at a time.
- Style plugins affect the system prompt and persist across conversations while active.
