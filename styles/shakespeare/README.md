# Shakespearean Mode 🎭

Hark! Doth transform RAGdoll into the Bard himself.

## What it does

Applies a system prompt prefix that instructs the AI to respond entirely in Early Modern English (Shakespearean style) — thou, thee, forsooth, and all the dramatic flair of the Globe Theatre.

## Usage

1. Install the plugin from the Marketplace.
2. In the **Installed** tab, click **Set active** next to Shakespearean Mode.
3. All subsequent responses will be delivered in Elizabethan English.
4. Click **Active** again to deactivate.

## Example

> **User:** Explain how neural networks work.  
> **Assistant:** Hark, good soul! Methinks thou dost seek to understand the wondrous artifice known as a neural network. Verily, 'tis a computational construct modelled upon the very sinews of the human mind…

## Notes

- Only one style plugin can be active at a time.
- Works best with models that have strong instruction-following (GPT-4o, Claude 3+).
- Technical accuracy is always preserved — only the delivery changes.
